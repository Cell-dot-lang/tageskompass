Einstellungen & Datenschutz

App

Apple-Kalender

Der geschützte Kalender-Worker liest deine privaten iCloud-Kalender. Apple-ID und App-Passwort werden niemals in dieser App oder auf GitHub gespeichert.

Der Zugriffsschlüssel wird nur auf diesem Gerät gespeichert und nicht in Backups exportiert. Deine sieben iCloud-Kalender bleiben privat.

Optionale KI-Auswertung

Ohne Einstellung läuft die private Offline-Auswertung. Ein eigener sicherer Server-Endpunkt kann bessere Formulierungen erzeugen. Trage hier niemals einen API-Schlüssel ein.

Daten löschen

Journal, Aufgaben und Auswertungen werden im Browser gespeichert. Ein Browserdaten-Löschen kann diese Daten entfernen. Regelmäßige Backups sind sinnvoll.
